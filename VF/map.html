Atlas.js Mapping Library Demo

Tile Layers

Base map tiles with attribution control

new Atlas.TileLayer(
  'https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png',
  { attribution: '© OpenStreetMap contributors' }
)

Markers

Interactive map markers with popups

new Atlas.Marker([51.5, -0.09], {
  title: 'London',
  popup: 'Welcome to London!',
  draggable: true
})

GeoJSON Layers

Vector shapes rendered on the map

new Atlas.GeoJSONLayer([
  {
    type: 'polygon',
    vertices: [[51, -0.1], [51.5, 0.1], [51.2, 0.3]],
    options: { fill: '#ff000055' }
  }
])

Popups

Customizable information windows

const popup = new Atlas.Popup()
  .setContent('Hello World!')
  .openOn(map, [51.5, -0.09])

Controls

Built-in UI components

new Atlas.Control.Attribution({
  prefix: 'Atlas.js Demo'
})

Navigation

Pan, zoom, and fit bounds

// Set view
map.setView([51.5, -0.09], 13)

// Fit bounds
map.fitBounds([[27, -13], [36, -1]])
