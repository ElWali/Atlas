Atlas.js Demo

A lightweight, single-file JavaScript mapping library inspired by Leaflet v0.1

Atlas.js Features

  • ✅ Lightweight single-file library (~15KB unminified)
  • ✅ Tile layer support with OpenStreetMap
  • ✅ Custom markers with built-in styling
  • ✅ Mouse pan and zoom interactions
  • ✅ Zoom controls
  • ✅ Spherical Mercator projection (EPSG:3857)
  • ✅ Modern ES6+ JavaScript
  • ✅ No dependencies

Usage Example

// Create a map
const map = Atlas.map('map').setView([51.505, -0.09], 13);

// Add a tile layer
Atlas.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png').addTo(map);

// Add a marker
Atlas.marker([51.5, -0.09]).addTo(map);
